# Tinkoff Workshop

1. Установить [Python 3.10](https://www.python.org/downloads/)
1. Установить [PyCharm](https://www.jetbrains.com/pycharm/download/)
1. Создать fork [репозитория](https://github.com/gurgenXD/tinkoff_workshop)
1. Склонировать репозиторий себе на ноутбук
1. Создать ветку с названием задачи
1. Сделать задачу
1. Создать Pull request
1. После ревью вмержить ветку в master
